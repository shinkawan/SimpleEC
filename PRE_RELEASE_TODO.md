# Simple EC v5.0.0 本番リリース前 最終点検・品質向上計画

本番リリースに向けて、システムの信頼性、セキュリティ、およびパフォーマンスを高い水準で確保するための包括的な点検とクリーンアップを実施します。

## ご確認事項

> [!IMPORTANT]
> この作業には、開発中に使用した一時的なスクリプトやデータの削除が含まれます。
> 以下の点についてご確認ください：
> 1. **一時スクリプトの削除**: `scratch/` フォルダ内のファイル（health-check等）はすべて削除します。
> 2. **エラーログのクリア**: 点検完了後、テスト中に溜まった古いエラーログをリセットします。

## 実施内容

### 1. セキュリティ・監査
各コンポーネントの管理者アクションおよびAJAX処理において、セキュリティ対策が万全であることを再確認し、必要に応じて強化します。
- `order-manager.php` などの主要ファイルのセキュリティスキャン。

### 2. コード・クリーンアップ
開発時の「形跡」を完全に消去し、軽量でプロフェッショナルな状態にします。
- `scratch/` フォルダ配下の全ファイル削除。
- `var_dump`, `print_r`, `console.log` の検索と除去。

### 3. デフォルト設定の適正化
- かご落ち遅延のデフォルト値が `24` となっているかコードレベルで最終確認。

### 4. ドキュメント同期
- `manual.html` に最新機能（カゴ落ち等）の説明を追記。
